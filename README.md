## Contenido del repositorio
Este repositorio consta una colección de soluciones en pocas líneas de código a problemas de programación planteados en la web [Acepta El Reto](https://www.aceptaelreto.com/). Todos ellos están programados en *C++* y validados como **Correct** por el juez automático de la web, siendo por tanto todas las soluciones suficientemente eficientes.
